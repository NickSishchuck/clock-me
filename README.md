# **CLI tool to track and display your working time**

## Commands:

Initialize the project you're working on
```bash
clock-me init
```
Start the working session
```bash
clock-me start
```
Start a break. You can continue working by typing "clock-me start"
```bash
clock-me break
```
Show the current status
```bash
clock-me status
```
End the working session
```bash
clock-me stop
```


## Installation:

```bash
cd your/folder

```
